# Privacy Policy — Cove

_Last updated: 2026-05-29_

## Summary

**Cove collects no data.** It does not track you, transmit information, or make any network requests.

## What Cove is

Cove is a Claude Code plugin consisting solely of a skill (`SKILL.md`) and a slash command (`cove.md`) — plain instructions that Claude follows locally within your Claude Code session. It contains no MCP servers, no executables, no scripts, and no third-party services.

## Data collection

- **Personal data collected:** None.
- **Usage analytics or telemetry:** None.
- **Network requests made by the plugin:** None.
- **Data shared with third parties:** None.

Any input you provide when using Cove (for example, the claim or code you ask it to verify) is handled entirely within your own Claude Code session and is subject to the privacy practices of Claude Code and Anthropic — not this plugin. Cove neither stores nor transmits that input itself.

## Changes

If this policy changes, the updated version will be published at this URL with a revised "Last updated" date.
